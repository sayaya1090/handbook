## 시나리오
### 데이터 모드
#### 타입 추가
    1. 빈 영역에서 컨텍스트 메뉴를 띄운다.
    2. 타입 추가 버튼을 클릭
    3. 타입명 입력
    4. 확인 버튼 클릭
    5. 화면에 타입이 추가된다.
       1에서 선택한 위치에 추가된다.
       타입 width에 의해 기존 그래프와 겹치는 부분이 생기면 기존 그래프를 밀어낸다.
#### 타입 삭제
    1. 박스를 선택한다.
    2. 컨텍스트 메뉴를 띄운다.
    3. 삭제 버튼을 클릭한다.
    4. 박스가 삭제된다.

#### 속성 추가
    1. 박스의 속성 추가 버튼을 클릭한다.
    2. 속성명과 타입을 입력한다.
    3. 확인 버튼을 클릭한다.
    4. 박스에 속성이 추가된다.
