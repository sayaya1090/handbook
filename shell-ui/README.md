### 동작
로딩된 스크립트(서브모듈)에서는 자신이 처리할 수 있는 페이지의 URL을
정의하고, 전역변수인 URL 변경을 감지하여 URL마다 렌더 함수를 재정의하는
방식으로 구현한다.

쉘에서 URL이 변경됨을 감지하면 짧은 디바운스 후 프레임 엘리먼트 파라미터를 
인자로 전역 함수인 렌더 함수를 호출한다.

메뉴 버튼이 클릭되면 해당 메뉴의 스크립트를 동적으로 로딩하고 렌더 함수를 
호출한다.
